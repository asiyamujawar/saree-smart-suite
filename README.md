# 🛍️ Mangalmurti Saree Management System

## 📌 Project Description

Mangalmurti Saree Management System is a modern web application designed to manage saree inventory, billing, and business operations efficiently. It is built using React, Vite, and TypeScript, with Supabase as the backend. The system includes a clean UI, dashboards, reports, and PDF invoice generation to provide a complete digital solution for saree businesses.

---

## ⚙️ How to Run the Project

### 1. Clone the repository
```sh
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
2. Install dependencies
npm install
3. Setup environment variables

Create a .env.local file in the root folder and add:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
4. Run the project
npm run dev

👉 Open in browser:

http://localhost:5173
